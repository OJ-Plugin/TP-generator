# 🧠 TP-Generator v1.2.2 Release Notes

**TP-Generator** is a desktop application tailored for **teachers** who need to create high-quality test points for online judge (OJ) platforms. By leveraging AI, it allows you to generate test data through two smart methods—helping automate and simplify your problem-setting workflow.

- 🐞 **Bug Fixes & Performance Optimizations:** Use AI service to generate input data and source code (in question to test point page but only available in this page), then use judge0 service to get output data with source code and input data in order to get more precise results.
